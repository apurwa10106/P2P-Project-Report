## P2P – Projects to Paychecks

###  Tagline:
**Turn Your Projects Into Real Income**

---

##  Overview

P2P (Projects to Paychecks) is a student-focused platform that helps transform academic and personal projects into real-world income opportunities.

Many students build projects but fail to monetize them or gain meaningful outcomes. P2P bridges this gap by providing:
- Monetization strategies
- Pricing models
- Company interest simulation
- Resume analysis
- Profile insights

---

##  Problem Statement

Students often:
- Build projects without real-world application
- Lack guidance on monetization
- Don't understand how to convert skills into income

---

##  Solution

P2P provides a structured journey:

1. Submit your project  
2. Get monetization & pricing strategy  
3. See companies interested  
4. Analyze your resume  
5. Get complete profile insights  

---

##  User Flow

Landing Page → Submit Project → Dashboard → Resume Analyzer → Profile Insights

---

##  Key Features

###  Monetization Strategy
Suggests ways to earn from the project (SaaS, freelancing, ads, etc.)

###  Pricing Plan
Provides tier-based pricing models based on project type

###  Income Target
Helps users define earning goals

### 🏢 Company Interest Simulation
Generates 20 relevant companies based on project domain

###  Resume Analyzer
Analyzes PDF resume and suggests:
- Role
- Score
- Career direction

###  Profile Insights
Displays:
- Skill level
- Career path
- Growth suggestions

---

##  Tech Stack

- **HTML** → Structure  
- **CSS** → Styling & UI  
- **JavaScript** → Logic & interactivity  
- **PDF.js** → Resume parsing  
- **LocalStorage** → Data handling  

---

##  How It Works

- User inputs project details
- JavaScript analyzes keywords (AI, Web, App)
- Based on domain:
  - Monetization strategy is generated
  - Pricing is assigned
  - Companies list is created (randomized)
- Resume is analyzed using keyword detection
- Final insights are displayed
